Detailed Description

Peça de tabuleiro.

Uma peça de tabuleiro base, contendo o ID do jogador dono da peça e o símbolo gráfico para desenha-la no tabuleiro.

O ID do dono segue a seguinte convenção:

  • 0: Peça sem dono (neutra)
  • 1: Peça do jogador 1
  • 2: Peça do jogador 2
  • X: Peça do jogador X
