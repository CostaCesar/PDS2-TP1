Detailed Description

Classe estática que gera números aleatórios.

As funções aleatórias do C++ (Standard Library) geram números diferentes para a mesma seed em ambientes diferentes (Arch Linux, Ubuntu, Windows 10, Windows 11, MacOs, etc.)

Para manter a consistência entre esses ambientes, foi criada essa classe
